Probability Logic

  • The logic will allow:

    • Dropping packets
    • Stable delay (packets keep the order in which they were received)
    • Unstable delay (packets may arrive out of order in the destination)
    • The behaviour of the arbitration will change in time, in order to simulate real-life networks
    • Implemented using several parallel Markovian chains, each with its own packet filter
